Restorator Support

Bome Software provides support for Restorator in multiple levels:

Tutorials (Usage Examples)

The Tutorials are examples of typical usage scenarios. Step-by-step guides with annotated screenshots make it easy to follow them by yourself.

Documentation (Reference Manual)

The Reference Manual provides descriptions of all features and functions offered by Restorator.

FAQ

The Frequently Asked Questions section provides ansers to the most often encountered problems.

Discussion Q&A Forums [online]

In the Q&A discussion forum [online], users can interchange know-how, tricks and their experiences. This is the recommended way to get help for specific problems.
The Bome team is active in the forums and makes sure that everybody's request will be satisfied.

Email Support [online]

Still can't solve your problem? Contact us by email with the contact form [online]!
Email support is reserved for licensed users (i.e. only if you have purchased Restorator).
